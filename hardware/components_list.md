# Components List

## Core Components

- ESP32 Dev Board
- Breadboard
- Jumper wires
- USB cable (for programming and power)

## Output Devices (Simulation)

- LEDs (x6)
  - 4 LEDs used to simulate lights
  - 2 LEDs used to simulate sockets

## Electrical Components

- Resistors (220Ω) for each LED

## Optional Components (For Real Deployment)

- Relay modules (for controlling AC appliances)
- External power supply
- Real appliances (lights, fan, sockets)

## Notes

- LEDs are used for safe testing and simulation
- Relays are required when controlling high-voltage devices
- Always ensure proper insulation and safety when working with AC power
